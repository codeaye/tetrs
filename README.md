## Run Locally

Clone the project

```bash
  git clone https://github.com/codeaye/tetrs/
```

Go to the project directory

```bash
  cd tetrs
```

Install dependencies

```bash
  cargo install just
```

Build for your platform

```bash
  just build
```

## Documentation

- Left arrow -> Move Left
- Right arrow -> Move right
- Up arrow -> Rotate
- Down arrow -> Soft drop
